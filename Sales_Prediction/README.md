# Sales Prediction using Machine Learning
## Overview
This project focuses on predicting product sales based on advertising expenditure across different marketing channels. Businesses often invest in advertising through platforms such as TV, radio, and newspapers. By analyzing historical advertising data, a machine learning model can learn the relationship between advertising budgets and product sales.

The goal of this project is to build a **sales prediction model using Python and machine learning techniques**.

## Objective
The objective of this project is to:
* Analyze advertising data
* Understand the relationship between advertising spending and product sales
* Train a machine learning model to predict future sales
* Evaluate the performance of the prediction model

## Dataset
The dataset used in this project contains advertising budgets and corresponding sales values.

### Dataset Features
| Feature   | Description                                   |
| --------- | --------------------------------------------- |
| TV        | Advertising budget spent on TV                |
| Radio     | Advertising budget spent on Radio             |
| Newspaper | Advertising budget spent on Newspapers        |
| Sales     | Sales generated from the advertising campaign |

## Technologies Used
The project is implemented using the following tools and libraries:
* **Python**
* **Pandas** – Data analysis and manipulation
* **NumPy** – Numerical computations
* **Matplotlib & Seaborn** – Data visualization
* **Scikit-learn** – Machine learning algorithms

## Project Workflow
### 1. Data Loading
The dataset is loaded using the Pandas library.

### 2. Data Exploration
Basic data analysis is performed to understand dataset structure, feature distribution, and relationships between variables.

### 3. Data Visualization
Visualization techniques such as scatter plots and pair plots are used to analyze how advertising expenditure affects sales.

### 4. Feature Selection
Advertising budgets (`TV`, `Radio`, `Newspaper`) are selected as input features, while `Sales` is used as the target variable.

### 5. Train-Test Split
The dataset is divided into:
* **80% training data**
* **20% testing data**

### 6. Model Training
A **Linear Regression model** is used to train the sales prediction system.

### 7. Model Evaluation
The model performance is evaluated using the **R² Score**, which measures how well the model predicts sales.

## Model Performance
The trained model achieves a high R² score, indicating a strong relationship between advertising expenditure and sales prediction.
