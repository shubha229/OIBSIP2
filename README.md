## OIBSIP Internship Projects
This repository contains the projects completed as part of the Oasis Infobyte Data Science Internship (OIBSIP). The goal of these projects is to apply data science and machine learning techniques to solve real-world problems.

The projects included in this repository focus on data preprocessing, exploratory analysis, machine learning model building, and prediction tasks.

Projects Included
### 1. Car Price Prediction
This project focuses on predicting the selling price of used cars using machine learning techniques. The model analyzes factors such as present price, kilometers driven, fuel type, transmission type, and ownership details to estimate the selling price. A Random Forest Regression model is used to learn patterns from the dataset and generate accurate price predictions.

### 2. Email Spam Detection
This project builds a classification model to detect whether an email message is spam or not. Natural Language Processing (NLP) techniques are used to process email text, convert it into numerical features, and train a machine learning model that can automatically identify spam messages.

### 3. Sales Prediction
This project develops a machine learning model to predict future sales based on advertising expenditure across different marketing channels such as TV, radio, and newspapers. By analyzing the relationship between advertising investment and sales performance, the model helps estimate expected sales outcomes.

These projects demonstrate practical implementation of Python, Pandas, NumPy, Scikit-learn, and data visualization libraries for building predictive models.

Each project folder contains the dataset, Jupyter Notebook implementation, and detailed project explanation.
